Workflow

Use this editor to author `assets/pixel-src/<assetId>.json`, then run `npm.cmd run tool:pixel:build -- --asset <assetId>` or `npm.cmd run tool:pixel:build:all`.

Pencil and erase respect mirror toggles. PNG import rescales and centers source art into the fixed 32x32 canvas.
